What's New in Version 1.12

• Improved sign-in reliability with automatic retry for connection issues
• Better error messages when network connection fails
• Fixed issue where free lifetime access users were incorrectly shown subscription prompts
• Enhanced weather data processing for more precise daily risk assessments
